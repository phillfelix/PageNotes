PagenoteJS

Um widget jQuery para gerar screenshots e anotações de páginas web

Simples utilização:

<script src="js/jquery.js"></script>
<script src="js/easeljs-0.7.1.combined.js"></script>
<script src="js/pagenote.min.js"></script>

Para testar, clique no botão 'Shot' no canto inferior esquerdo da página!
